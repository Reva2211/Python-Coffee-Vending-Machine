#Python-Coffee-Vending-Machine


A simple Python mini-project that simulates the working of a coffee vending machine.  
This project is beginner-friendly and demonstrates the use of dictionaries, functions, loops, and condition handling in Python.

---

## 🚀 Features

- Order different coffee types:
  - Espresso
  - Latte
  - Cappuccino
- Ingredient availability checking
- Coin-based payment system
- Automatic change calculation
- Resource management
- Profit tracking
- Machine report generation
- Machine shutdown option

---

## 🛠 Technologies Used

- Python 3

---

## 📚 Concepts Covered

- Nested Dictionaries
- Functions
- Loops
- Conditional Statements
- Global Variables
- User Input
- Resource Management Logic

---

## ▶️ How It Works

1. User selects a coffee type.
2. Program checks ingredient availability.
3. User inserts coins.
4. Payment is verified.
5. Coffee is prepared if payment is successful.
6. Resources get updated automatically.

---

## 💰 Coin System

The machine accepts:
- ₹5 coins
- ₹10 coins
- ₹20 coins

---

## 📄 Commands

| Command | Description |
|---------|-------------|
| latte | Order Latte |
| espresso | Order Espresso |
| cappuccino | Order Cappuccino |
| report | Display available resources and profit |
| off | Turn off the machine |
